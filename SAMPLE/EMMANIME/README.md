# EMMのデータでアニメーション

　EMMに格納されているデータをG-RAMに転送することでアニメーションしようというプログラムです。

　MZ-2000用とMZ-80B用と２種類あるので機種にあった方を使ってください。

## EMMANIMEの使い方
　EMMにデータを転送したのちに機種とEMMに合わせて以下からプログラムを選択し、実行すればアニメーションが始まります。

　止めたいときには何かキーを押せばモニターに戻ります。

### MZ-2000用プログラム
　SAMPLEフォルダ、EMMANIMEフォルダ、EMMANIME2000内のEMM0ANIME2000.mzt、EMM1ANIME2000.mzt、EMM2ANIME2000.mzt、EMM3ANIME2000.mztのいずれかを使います。

　MZ-2000_SDを使っていればSDカードにコピーします。

### MZ-80B用プログラム
　SAMPLEフォルダ、EMMANIMEフォルダ、EMMANIME80B内のEMM0ANIME80B.mzt、EMM1ANIME80B.mzt、EMM2ANIME80B.mzt、EMM3ANIME80B.mztのいずれかを使います。

　MZ-2000_SDを使っていればSDカードにコピーします。

## データ
　以下にフォーマットを示しますが、枚数分の画像データにmztファイル用のヘッダが付加された形となっています。

　何らかの方法で画像データを作成しそれらを一つのファイルにまとめてEMMへコピーします。

　EMMへのコピーはEMMLOADで行います。

### MZ-2000
|データ長|内容|備考|
| ------------ | ------------ | ------------ |
|0080h|mztファイルヘッダ|00Hを128Byteで可|
|4000h|PAGE01画像データ||
||[30枚分の画像データ]||
|4000h|PAGE32画像データ||

　サンプルとしてPIO3034_1_anime2000.MZTを置いてあります。

### MZ-80B
|データ長|内容|備考|
| ------------ | ------------ | ------------ |
|0080h|mztファイルヘッダ|00Hを128Byteで可|
|2000h|PAGE01画像データ||
||[62枚分の画像データ]||
|2000h|PAGE64画像データ||

　サンプルとしてPIO3034_1_anime80b.MZTを置いてあります。

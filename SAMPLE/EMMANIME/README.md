# EMMのデータでアニメーション

　EMMに格納されているデータをG-RAMに転送することでアニメーションしようというプログラムです。

　MZ-2000とMZ-80Bを自動判別して起動します。

## EMMANIMEの使い方
　EMMには先にデータを作成または転送しておく必要があります。

　EMMANIME.EXEを実行すると実行環境がMZ-2000か、MZ-80Bか自動判別して起動します。

　どのEMMのデータを使うか選択すればそのデータによりアニメーションが始まります。

　止めたいときには何かキーを押せばモニターに戻ります。

## データ
　以下にフォーマットを示しますが、枚数分の画像データにmztファイル用のヘッダが付加された形となっています。

　何らかの方法で画像データを作成しそれらを一つのファイルにまとめてEMMへコピーします。

　MZ-2000_SDを使用していればEMMMENUを使用してEMMへデータをコピーすることが出来ます。

### MZ-2000
|データ長|内容|備考|
| ------------ | ------------ | ------------ |
|0080h|mztファイルヘッダ|00Hを128Byteで可|
|4000h|PAGE01画像データ||
||[30枚分の画像データ]||
|4000h|PAGE32画像データ||

　サンプルとしてPIO3034_1_anime2000.MZTを置いてあります。

### MZ-80B
|データ長|内容|備考|
| ------------ | ------------ | ------------ |
|0080h|mztファイルヘッダ|00Hを128Byteで可|
|2000h|PAGE01画像データ||
||[62枚分の画像データ]||
|2000h|PAGE64画像データ||

　サンプルとしてPIO3034_1_anime80b.MZTを置いてあります。
